URL: http://habariproject.org
Version: 1.0
Author: Habari Project


# Purpose

AutoApprove will automatically approve comments that have not been flagged as spam.


# Requirements 

None.


# Installation

1. Copy the plugin directory into your user/plugins directory or the site's plugins directory.
2. Go to the plugins page of your Habari admin panel.
3. Click on the Activate button for AutoApprove.


# Uninstallation

1. Got to the plugins page of your Habari admin panel.
2. Click on the Deactivate button.
3. Delete the 'autoapprove' directory from your user/plugins directory.
